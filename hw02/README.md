# DeepSeek-R1 Chatbot 运行说明

本项目基于火山引擎平台调用DeepSeek-R1大模型实现简单的Chatbot功能，可实现问题查询并能输出模型专属的推理过程，以下是详细的运行、配置与使用说明。

## 运行环境

- 操作系统：Windows/Linux/macOS 均可
- Python 版本：3.8及以上（推荐3.9-3.11，保证第三方库兼容性）
- 网络环境：可正常访问火山引擎API地址（https://ark.cn-beijing.volces.com）

## 依赖安装

项目仅需安装OpenAI SDK，用于兼容火山引擎的API调用规范，执行以下命令完成安装：

```bash
# 安装基础版本，满足调用需求
pip install openai
# 推荐安装最新版本，提升兼容性
pip install --upgrade openai>=1.0
```

**验证安装**：执行以下命令，若能显示openai的版本、路径等信息，说明安装成功

```bash
pip show openai
```

## API Key 与配置项设置

### 前置准备

需先在火山引擎平台完成DeepSeek-R1模型的开通、接入点创建，并获取**API Key**和**模型接入点ID**（格式为`ep-时间戳-随机码`
），具体操作参考火山引擎官方指引。


```python
# 替换为你自己的火山引擎API Key
api_key = "你的API Key"
# 替换为你创建的DeepSeek-R1模型接入点ID
model = "你的模型接入点ID"
```

import os

api_key = os.environ.get("ARK_API_KEY")
```

**模型接入点ID**：需在代码的`model`参数中手动替换为自己的ID，无环境变量适配方式。

## 运行命令

1. 将代码保存为Python文件，例如`deepseek_chatbot.py`
2. 打开终端/CMD，进入文件所在目录，执行以下命令运行：

```bash
python deepseek_chatbot.py
```

## 常见问题排查

1. **API Key错误**：报错提示密钥无效，检查是否复制正确，或是否在火山引擎平台开通了DeepSeek-R1模型的使用权限。
2. **模型接入点ID错误**：报错提示模型不存在，确认ID与创建的DeepSeek-R1接入点完全一致，格式为`ep-xxxx-xxxx`。
3. **网络问题**：无法连接API地址，检查网络是否通畅，是否需要配置代理。
4. **无推理过程输出**：部分简单问题模型不会触发深度推理，属于正常现象，仅会输出最终答案。
5. **依赖版本问题**：报错提示`chat.completions.create`不存在，升级openai版本至1.0及以上即可。
